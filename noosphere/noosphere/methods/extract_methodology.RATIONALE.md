# extract_methodology

Noosphere previously extracted object-level claims and only incidentally
classified a few as methodological. That lost the reusable part of serious
dialogue: the way a conclusion was reached.

This method produces deterministic methodology profiles from source text. The
profiles are intentionally not final judgments. They are structured first-pass
records that make later review, publication, and cross-domain transfer ask:

- What reasoning move is being used?
- What assumption makes the move work?
- Where else might the move transfer?
- Where would this method fail?
- Which source sentences justify the profile?

The method is deterministic because it is used in reanalysis and backfills. A
future LLM pass can improve the text, but it should write into the same
profile-shaped contract rather than inventing a new surface.

The transfer targets describe reusable arenas for the method, not the source
topic itself. For example, an education transcript may yield institutional
design as a transfer target, but the education conclusion is preserved only as
source evidence, not as a portable answer.
