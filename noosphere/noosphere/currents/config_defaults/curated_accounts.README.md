# Curated X accounts

Populate `curated_accounts.json` with X handles (without the leading `@`) whose posts should be ingested as candidate current events.

Curation guideline: only include accounts whose claims are substantive enough to warrant firm commentary — journalists, domain experts, primary sources. Hot-take accounts create noise and drain the opinion-generation budget on events the firm cannot meaningfully opine on.

An empty list disables the curated-account path entirely.
