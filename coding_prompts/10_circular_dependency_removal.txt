Implement:

A. Detection. Run `pydeps` (or equivalent) to produce the import

B. Resolve cycles by introducing interfaces. A cycle A ↔ B is

   move the import to function-local. Each function-local import

D. Type-only edges. In TypeScript, prefer `import type` for
