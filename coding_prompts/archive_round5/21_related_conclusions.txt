## The problem

Each conclusion exists in isolation on its detail page. There is no way to see other conclusions that are epistemically connected — conclusions drawn from the same source material, conclusions that share supporting principles, or conclusions whose claims appear in this conclusion's evidence chain or dissent list. Understanding the network of belief is impossible one conclusion at a time.

## Your task

Add a "Related conclusions" section to the conclusion detail page that shows other conclusions connected to the current one via shared sources or shared principle/claim IDs.

   **Method A — shared source uploads:** Query `ConclusionSource` to find other conclusions that share at least one source upload with this conclusion:
   ```typescript
   // Get this conclusion's source upload IDs
   const sources = await db.conclusionSource.findMany({
     where: { conclusionId },
     select: { uploadId: true },
   });
   const uploadIds = sources.map((s) => s.uploadId);

   // Find other conclusions from the same uploads
   const relatedSources = await db.conclusionSource.findMany({
     where: {
       uploadId: { in: uploadIds },
       conclusionId: { not: conclusionId },
     },
     include: {
       conclusion: { select: { id: true, text: true, confidenceTier: true, confidence: true } },
     },
     distinct: ["conclusionId"],
     take: 20,
   });
   ```

Search for other conclusions that contain any of the same IDs in their respective arrays.
