## The problem

The provenance tab shows `extractionMethod` as raw text (e.g., "deduction", "induction", "empirical"). There is no explanation of what these method names mean, and no link to the method registry where the user could learn more about the method's definition, version, and usage statistics.

## Your task

Make the extraction method name a clickable link to the method registry entry. If the methods page uses URL patterns like `/methods?name=deduction`, link there. If no methods page exists, link to nothing but add a tooltip explaining the method.
