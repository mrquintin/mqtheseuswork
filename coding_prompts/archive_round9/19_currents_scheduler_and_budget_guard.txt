### Status file

After every cycle, write `noosphere_data/currents_status.json`:

```json
{ "last_cycle": { ...CycleReport... } }
```

The FastAPI service's `/readyz` (prompt 10) checks this file's mtime; if older than 600s (2× cycle), service is marked not-ready.

### `__main__.py`

`python -m noosphere.currents once` and `python -m noosphere.currents loop` are operator entry points.

### Budget persistence wiring

The API service (prompt 10) instantiates `HourlyBudgetGuard.load(path)` at startup. Both processes (scheduler + API) read/write the same JSON. Atomic `tempfile + os.replace`. Hour rollover resets to zero — never carry balances.

### In-process publisher

The API service tails the DB cursor (`OpinionTailer` from prompt 10) so SSE subscribers get new opinions in near-real-time after the scheduler writes them. No additional broker.
