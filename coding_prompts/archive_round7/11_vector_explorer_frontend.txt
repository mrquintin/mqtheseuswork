## The problem

The founder wants to "visualize the conclusions themselves" by plotting them in the vector embedding space on semantic axes. This requires an interactive 2D scatter plot where:

1. Each dot represents a conclusion.
2. The axes represent the principal semantic dimensions computed by the backend (prompt 10).
3. Founders can hover over or click dots to see the conclusion text.
4. Visual encoding communicates metadata: confidence tier (color), topic (shape or color), etc.

2. **Create the scatter plot client component** at `theseus-codex/src/app/(authed)/explorer/scatter-plot.tsx`:
   ```tsx
   "use client";
   ```

   This component should:
   - Fetch `/api/conclusions/embeddings` on mount.
   - Render a 2D scatter plot using SVG (no heavy library dependency needed — the dataset is small enough for raw SVG).
   - Show a loading state while fetching.
   - Handle the error case (< 3 conclusions) with a message.

3. **Scatter plot implementation details:**

   **Layout:**
   - The plot area should be responsive, roughly 800×600px with margins for axis labels.
   - Use SVG `<circle>` elements for each conclusion.
   - Scale x/y coordinates to fit the SVG viewBox using linear scaling from the min/max of each projected dimension.

   **Visual encoding:**
   - Circle color by confidence tier:
     - `firm` → `var(--gold)` or `#d4a017`
     - `founder` → `var(--amber)` or `#c9944a`
     - `open` → `var(--parchment)` or `#c8b89a`
     - `speculative` → `var(--parchment-dim)` or `#8a7e6b`
   - Circle radius: 5px default, 8px on hover.
   - Circle opacity: 0.7 default, 1.0 on hover.

   **Interactivity:**
   - **Hover:** Show a tooltip with the conclusion text (truncated to ~200 chars), confidence tier, and topic hint.
   - **Click:** Navigate to `/conclusions/{id}` for the full detail page.
   - **Tooltip positioning:** Position near the cursor, clamped to the SVG bounds.

   **Axes:**
   - Draw axis lines through the center of the plot.
   - Label each axis with the `SemanticAxis.label` from the API response.
   - Show the `varianceExplained` percentage next to each axis label.
