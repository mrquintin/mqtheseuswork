Goal: monitor new empirical situations by applying abstract principles learned from prior cases.

Product direction:
When Noosphere encounters a new event, market, company, institution, or case, it should not only retrieve topically similar source material. It should ask: which abstract principles apply, which preconditions are present, which failure conditions are present, which prior cases are structurally analogous, and what decision follows under each applicable framework.

Implement:
1. Add an analogical transfer engine:
   - input: new case/event/market;
   - retrieve candidate principles;
   - retrieve prior cases;
   - score structural similarity, not only keyword similarity;
   - check preconditions;
   - check failure conditions;
   - return transfer recommendations.
2. Add transfer metrics:
   - structural fit;
   - missing preconditions;
   - contradiction risk;
   - case distance;
   - domain shift;

   - mechanism match;
