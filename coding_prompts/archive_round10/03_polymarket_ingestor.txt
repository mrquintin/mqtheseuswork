### `polymarket_ingestor.py`

```python
async def ingest_once(store: Store, *, config: PolymarketConfig,
                      now: datetime | None = None) -> IngestResult:
    """
    Pull active+open markets in batches, upsert each into ForecastMarket.

    Dedupe key: (source=POLYMARKET, externalId=conditionId).

    For each row:
      - parse title, description, resolutionCriteria from the Gamma payload
      - extract currentYesPrice / currentNoPrice from outcome[*].price
      - parse openTime + closeTime
      - assign category from `tags[]` ∩ accepted_categories (or first tag if empty)
      - serialize the entire raw payload into ForecastMarket.rawPayload
      - upsert; bump updatedAt only when *price* changed by >= 0.005 OR
        title/closeTime/status changed; otherwise skip the write to keep
        downstream change-detection tight.
    """

@dataclass
class IngestResult:
    fetched: int
    inserted: int
    updated: int
    skipped: int
    errors: list[str]
```

Filter rules baked in:
- skip markets where `closed == True`
- skip markets where `closeTime < now` (already past resolution)
- if `accepted_categories` is non-empty, drop markets whose tags do not intersect
