Goal: every safety property the firm has shipped — the eight-gate
contract, the trigger-predicate sandbox, the verbatim-citation
discipline, the no-real-network-in-tests rule, the no-PII-in-logs

for the safety properties to silently break.

The catalog of properties this suite protects:

P1. **Trigger-predicate sandbox is unescapable.**
   Adversarial predicates (`__import__('os').system(...)`,
   attribute access, list comprehensions over `__globals__`,
   etc.) MUST be rejected at parse / eval time. The runtime
   pauses the algorithm after 3 rejections.

P2. **Algorithm runtime makes zero unmocked network calls in
    tests.**
    Monkeypatch httpx.AsyncClient.request; count calls; assert
    0 across the full test suite for the algorithm package.

P3. **Memo PDF generation cannot execute arbitrary code.**
    pdflatex is run in a constrained mode (shell-escape DISABLED).
    A test attempts to plant `\immediate\write18{rm -rf /tmp/x}`
    in a memo body and verifies pdflatex refuses (or runs
    without executing the shell call).

P4. **Eight-gate safety contract stands.**
    Every gate from Round 10 prompt 09 still rejects under the
    expected condition. Specifically test against every Round 19
    bet kind (MARKET_BET, ADVISORY_BET, STRATEGIC_BET,
    SCIENTIFIC_BET) that MARKET_BET is the only kind that runs
    the gate check; the others have their own simpler controls.

P5. **Verbatim citation discipline holds.**
    Memos cite verbatim from principles; algorithm reasoning
    traces cite verbatim from invoked principles; the synthesizer's
    reasoning chain cites verbatim. Each invocation point gets a
    fabrication-fixture test: planted "almost verbatim" citations
    are rejected; exact substrings are accepted.

P6. **Provenance policy is honored.**
    A synthesis with `include_opposing_external=false` MUST NOT
    cite any OPPOSING_EXTERNAL source. Plant an adversarial
    LLM response that cites one anyway; verify the validator
    catches the violation and the result is rejected.

P7. **No PII / secret in any log output.**
    Marker values placed in env (`POLYMARKET_PRIVATE_KEY=
    0xMARKERVALUE`, `KALSHI_API_PRIVATE_KEY=PEM_MARKER`,
    `DATABASE_URL=postgresql://X_USER:X_PASS@...`) MUST NOT
    appear in any captured log when the full smoke harness
    runs.

P8. **Operator HMAC secret guards operator routes.**
    Every Round-19 operator-only route returns 403 without a
    valid HMAC; 200 with a valid one. Brute-force a wrong
    signature; assert 403 and structured-log entry.

P9. **Kill switch blocks every live path.**
    With `ForecastPortfolioState.killSwitchEngaged=true`, every
    attempted live bet across MARKET_BET subkinds (Polymarket,
    Kalshi, Alpaca, Robinhood) is refused with
    `KILL_SWITCH_ENGAGED`.

P10. **Idempotency holds end-to-end.**
     Running the algorithm runtime twice over the same input
     state produces one invocation, not two. Running the
     synthesizer twice on the same question + same state
     produces one memo, not two. Running portfolio dispatch
     twice on the same memo produces one MemoDispatch, not two.
