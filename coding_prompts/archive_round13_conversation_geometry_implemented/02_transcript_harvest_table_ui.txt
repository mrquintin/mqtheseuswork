## Task

Add a `ConversationGeometryPanel` on `/transcripts/[uploadId]` after the blurb and before the raw transcript. It must include:

- a Map view: SVG speaker nodes with weighted lines between speakers.
- a Weighting view: participant share, turns, question count, average turn length, and repeated terms.
- a Catalysts view: high-scoring point-to-point handoffs linking back to the exact target chunk.
- a year-end signal card that is locked for an open season/year.

Use the existing amber Codex visual system, but include a second restrained accent for graph edges/term chips so the panel does not become a one-hue amber wall.
