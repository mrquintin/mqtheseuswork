2. Build `reviewers/__init__.py` with a registry:

3. Build `swarm.py` with `SwarmOrchestrator(store)`:
     def run(self, conclusion_id: str) -> SwarmReport:
         reviewers = [cls() for cls in all_reviewers()]
         reports = run_in_parallel([r.review(conclusion, context) for r in reviewers])  # no shared state
         swarm_report = SwarmReport(conclusion_id=conclusion_id, reviews=reports, rebuttals=[])
         store.insert_review_reports(reports)
         return swarm_report
   Use `concurrent.futures.ThreadPoolExecutor`. Do NOT pass `reports` into subsequent reviewers — strict independence.
4. Build `rebuttal.py`:
