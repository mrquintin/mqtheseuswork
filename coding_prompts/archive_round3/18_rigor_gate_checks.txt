## Step-by-step
1. Create `checks/<name>.py` with a function `run(submission: RigorSubmission) -> CheckResult`, decorated as a registered method (name=f"gate_check_{name}").
2. At module bottom, call `rigor_gate.checks.register(name, run)`.
3. Guard upstream-subsystem imports in try/except; stub-pass on ImportError.
4. Write ≥ 3 tests per check: one passing, one failing, one stub-pass (achieved by monkey-patching the import to fail).

CHECK SPECIFICATIONS
provenance_complete.py
  For each load-bearing claim in submission payload, walk the cascade via `cascade.traverse.explain`; fail if any claim lacks a path to a corpus artifact or cited external source.

unresolved_honesty.py
  Parse confidence assertions in the payload; for each, look up the latest `CounterfactualEvalRun` for the producing method; fail if the payload reports a confidence that is not the calibration-discounted value (beyond a small tolerance).

swarm_clean.py
  For each firm-tier conclusion cited in the payload, call `peer_review.swarm.latest_report(conclusion_id)`; fail if there is any unresolved `blocker` finding.

freshness.py
  For every object referenced in the payload, call `decay.freshness.compute_freshness(object_id)`.
  public_site intended_venue requires all `fresh`.
  rss / api: `fresh | aging` with a visible banner marker included in conditions.
  stale / retired: always fail.

copyright_clean.py
  Scan payload text for verbatim substrings of ≥ 20 consecutive words from any ingested artifact. Fail on any hit.

tenant_isolation.py
  Collect every referenced object's `tenant_id`; fail if any differs from the submission's `author.tenant_id`.

personal_info_scrub.py
  Run PII filters (email regex, phone regex, address patterns, names not in a curated public-figures whitelist); fail on any hit not pre-declared in `author_attestation.acknowledgments`.

no_hidden_claim.py
  LLM-based: given the payload text and the set of traced claims, ask whether any assertion in the payload is not traceable. Fail if so.

dialectic_consent.py
  Only relevant when `submission.kind == "dialectic_summary"`. For every identified speaker, look up consent records; fail if any speaker lacks public-release consent at the published granularity.
