### What this involves:
- **macOS code signing**: `codesign` the .app bundle (Dialectic) and the CLI binary directory (Noosphere) with a Developer ID certificate.
- **macOS notarization**: Submit the signed artifact to Apple's notarization service via `notarytool`, staple the result.
- **Windows Authenticode signing**: `signtool` the .exe files with a code signing certificate.
- **Self-update check**: A lightweight update checker that hits a version manifest URL and notifies the user when a new version is available (no silent auto-install for PyInstaller apps — just notification + download link).

## Step-by-step

### macOS code signing and notarization

1. **Create `scripts/codesign_macos.sh`** — signs a PyInstaller output directory or .app bundle:
   ```bash
   #!/usr/bin/env bash
   # Usage: ./scripts/codesign_macos.sh <path-to-app-or-dir> <signing-identity>
   # If signing identity is empty or "skip", signing is skipped gracefully.
   set -euo pipefail

   TARGET="$1"
   IDENTITY="${2:-skip}"

   if [[ "$IDENTITY" == "skip" || -z "$IDENTITY" ]]; then
     echo "⚠ No signing identity provided — skipping code signing."
     exit 0
   fi

   echo "=== Signing $TARGET with identity: $IDENTITY ==="

   # Deep-sign all nested binaries first (required for notarization)
   find "$TARGET" -type f \( -name "*.dylib" -o -name "*.so" -o -perm +111 \) -exec \
     codesign --force --options runtime --timestamp --sign "$IDENTITY" {} \;

   # Sign the top-level bundle or directory
   codesign --force --deep --options runtime --timestamp --sign "$IDENTITY" "$TARGET"

   # Verify
   codesign --verify --deep --strict "$TARGET"
   echo "=== Signing complete ==="
   ```

2. **Create `scripts/notarize_macos.sh`** — submits a DMG or ZIP to Apple's notarization service:
   ```bash
   #!/usr/bin/env bash
   # Usage: ./scripts/notarize_macos.sh <path-to-dmg-or-zip> <apple-id> <team-id> <app-specific-password>
   set -euo pipefail

   ARTIFACT="$1"
   APPLE_ID="${2:-}"
   TEAM_ID="${3:-}"
   APP_PASSWORD="${4:-}"

   if [[ -z "$APPLE_ID" || -z "$TEAM_ID" || -z "$APP_PASSWORD" ]]; then
     echo "⚠ Notarization credentials not provided — skipping."
     exit 0
   fi

   echo "=== Submitting $ARTIFACT for notarization ==="
   xcrun notarytool submit "$ARTIFACT" \
     --apple-id "$APPLE_ID" \
     --team-id "$TEAM_ID" \
     --password "$APP_PASSWORD" \
     --wait --timeout 600

   echo "=== Stapling notarization ticket ==="
   xcrun stapler staple "$ARTIFACT"

   echo "=== Notarization complete ==="
   ```

### Windows code signing

3. **Create `scripts/codesign_windows.ps1`**:
   ```powershell
   # Usage: .\scripts\codesign_windows.ps1 -Target <exe-or-dir> -CertPath <pfx-path> -CertPassword <pw>
   param(
     [Parameter(Mandatory=$true)] [string]$Target,
     [string]$CertPath = "",
     [string]$CertPassword = ""
   )

   if (-not $CertPath -or -not (Test-Path $CertPath)) {
     Write-Host "⚠ No certificate provided — skipping code signing." -ForegroundColor Yellow
     exit 0
   }

   $signtool = Get-ChildItem "C:\Program Files (x86)\Windows Kits\*\bin\*\x64\signtool.exe" -Recurse | Select-Object -First 1

   if (-not $signtool) {
     Write-Host "⚠ signtool.exe not found — skipping." -ForegroundColor Yellow
     exit 0
   }

   # Find all .exe and .dll files
   $files = Get-ChildItem -Path $Target -Recurse -Include *.exe,*.dll

   foreach ($file in $files) {
     Write-Host "Signing $($file.FullName)..."
     & $signtool.FullName sign /f $CertPath /p $CertPassword /tr http://timestamp.digicert.com /td sha256 /fd sha256 $file.FullName
   }

   Write-Host "=== Windows signing complete ===" -ForegroundColor Green
   ```

### Self-update checker

8. **Create `dialectic/dialectic/updater.py`**:
   ```python
   """Lightweight update checker for Dialectic.

   On launch, checks a version manifest URL (JSON) for the latest version.
   If newer than the running version, shows a non-blocking notification in the UI.
   No silent auto-install — user must download manually.
   """
   import json
   import logging
   import threading
   from importlib.metadata import version as pkg_version
   from urllib.request import urlopen, Request
   from urllib.error import URLError
   from packaging.version import Version

   logger = logging.getLogger(__name__)

   UPDATE_MANIFEST_URL = "https://releases.theseus.co/dialectic/latest.json"
   # Manifest format: {"version": "0.2.0", "download_url": "https://...", "release_notes": "..."}

   def check_for_updates(callback=None, manifest_url: str = UPDATE_MANIFEST_URL):
       """Check for updates in a background thread.
       Calls callback(version_info_dict) if an update is available.
       Silently does nothing on network errors."""
       def _check():
           try:
               current = Version(pkg_version("dialectic"))
               req = Request(manifest_url, headers={"User-Agent": "Dialectic-Updater"})
               with urlopen(req, timeout=10) as resp:
                   data = json.loads(resp.read())
               latest = Version(data["version"])
               if latest > current and callback:
                   callback(data)
           except (URLError, KeyError, Exception) as e:
               logger.debug(f"Update check failed (non-fatal): {e}")

       t = threading.Thread(target=_check, daemon=True)
       t.start()
       return t
   ```
