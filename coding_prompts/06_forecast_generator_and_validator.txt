### `noosphere/noosphere/forecasts/forecast_generator.py`

```python
class ForecastOutcome(str, Enum):
    PUBLISHED = "PUBLISHED"
    ABSTAINED_BUDGET = "ABSTAINED_BUDGET"
    ABSTAINED_INSUFFICIENT_SOURCES = "ABSTAINED_INSUFFICIENT_SOURCES"
    ABSTAINED_NEAR_DUPLICATE = "ABSTAINED_NEAR_DUPLICATE"
    ABSTAINED_CITATION_FABRICATION = "ABSTAINED_CITATION_FABRICATION"
    ABSTAINED_MARKET_EXPIRED = "ABSTAINED_MARKET_EXPIRED"

async def generate_forecast(
    store, market_id: str, *, budget,
) -> ForecastOutcome:
    """
    Pipeline:
      1. Load ForecastMarket. If status != OPEN or closeTime < now+1h → ABSTAINED_MARKET_EXPIRED.
      2. retrieve_for_market(store, market, top_k=8). If <3 distinct sources → ABSTAINED_INSUFFICIENT_SOURCES.
      3. Check near-duplicate against existing PUBLISHED predictions for this org by
         cosine on the headline embedding; if cos > 0.92 with one in last 24h →
         ABSTAINED_NEAR_DUPLICATE.
      4. Authorize budget. If denied → ABSTAINED_BUDGET.
      5. Compose system prompt + user payload. PromptSeparator-wrap the user payload
         (delimits market metadata + sources block).
      6. Call Haiku 4.5 with strict JSON schema (see below). Up to 2 retries on JSON
         parse failure with corrective system message.
      7. Validate every citation: quoted_span MUST be a verbatim substring of the cited
         source.text. Up to 2 retries; persistent failure → ABSTAINED_CITATION_FABRICATION.
      8. Persist ForecastPrediction + ForecastCitation rows. Charge the budget with the
         actual prompt/completion tokens.

    Returns the ForecastOutcome.
    """
```

### Strict JSON schema enforced on the model

```jsonc
{
  "probability_yes": 0.0–1.0,
  "confidence_low": 0.0–1.0,
  "confidence_high": 0.0–1.0,        // must be >= probability_yes >= confidence_low
  "headline": "<= 140 chars",
  "reasoning_markdown": "<= 1800 chars",
  "uncertainty_notes": "<= 500 chars",
  "topic_hint": "<= 40 chars (snake_case)",
  "citations": [
    {
      "source_type": "CONCLUSION" | "CLAIM",
      "source_id": "<id>",
      "quoted_span": "exact substring of source.text, <= 240 chars",
      "support_label": "DIRECT" | "INDIRECT" | "CONTRARY"
    },
    ...
  ]
}
```

A `probability_yes` outside `[confidence_low, confidence_high]` triggers a single corrective retry; persistent failure → ABSTAINED_CITATION_FABRICATION (reuse the enum; the failure mode is "model produced nonsense"). Reasoning_markdown must contain at least one of the source_ids cited; otherwise treat as fabrication.
