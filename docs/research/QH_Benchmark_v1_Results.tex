\begin{abstract}We report the first end-to-end run of the Quintin Hypothesis (QH) benchmark v1 against the firm's contradiction-geometry probe and two baselines (a random runner and a cosine-only runner). The frozen v1 dataset contains 1936 items spanning 3 domains. All runners use the deterministic hash-det-v1 embedder, so the comparison is on geometry, not on embedding quality. \par\textbf{Honest negative finding.} On this run a non-firm baseline beats the firm's contradiction-geometry probe on at least one slice. The benchmark was built so the firm could be wrong in public; the losing slices are enumerated in Section~\ref{sec:honest}, not buried. The MQS-on-the-firm-probe composite is below the announcement threshold, so this result is published but not promoted.\end{abstract}

\section{Honest findings}\label{sec:honest}
A non-firm baseline wins on the following slices:
\begin{itemize}
\item Overall 3-way accuracy: the cosine baseline (0.3673) beats the firm's contradiction-geometry probe (0.2877).
\item Overall 3-way accuracy: even the random runner (0.3352) beats the firm probe (0.2877) --- the probe is below chance on accuracy.
\item Domain 'economics' accuracy: cosine (0.4094) beats the firm probe (0.2596).
\item Domain 'ethics' AUROC: cosine (0.5124) beats the firm probe (0.4667).
\item Domain 'physics' accuracy: cosine (0.4037) beats the firm probe (0.2898).
\item The firm probe is degenerate on this run: it predicts 'contradicting' for every item. Its AUROC is not meaningless, but its accuracy reflects only the base rate of that label.
\end{itemize}
