# Security probes — 2026-05-14 (Round 18 follow-up)

Owner: founders. Scope: Round 18 public surfaces. Environment:
staging. Source: `scripts/security_probe_staging.sh` plus targeted
manual review.

Production was **not** probed. The probe script refuses to run unless
`THESEUS_ALLOW_PROBE=staging` is exported; running it against
production additionally requires `THESEUS_ALLOW_PROBE=production-i-am-the-founder`,
which is a deliberately ugly step to prevent a misfire.

> **Sensitive findings policy.** If a probe produces an actual
> exposure (a leaked secret, a writable endpoint that should be
> read-only, a private record returned through a public route), it
> is **not** committed to git. It is messaged to the founder
> directly. This file contains only findings that are already in the
> public threat model.
