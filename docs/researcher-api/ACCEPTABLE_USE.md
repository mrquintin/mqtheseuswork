# Acceptable use — Theseus Researcher API

This policy applies to anyone holding a Theseus Researcher API key. Violations may result in **immediate key suspension**; appeals go to the same contact that issued the key.

## Allowed use

- Academic or journalistic research on **methodology**, belief revision, calibration, or coherence, using **your own** text, claims, corpora, and prediction sets.
- Replication studies that cite the API version and deployment metadata (see [CITATION.bib](./CITATION.bib)).

## Prohibited use

1. **Political opposition research at scale** — automated or bulk profiling of voters, officials, or campaigns for partisan targeting.
2. **Adversarial coherence against named individuals without consent** — e.g. mass generation of “incoherent” verdicts aimed at harassment, reputation attacks, or non-consensual psychological profiling.
3. **Commercial resale** of raw API outputs or a competing API that wraps Theseus, **without** a separate written license from the firm.
4. **Personal data you are not lawfully entitled to process** — do not submit highly sensitive personal information through the API.

## Enforcement

Suspension is keyed to the API credential. Researchers may receive a short explanation and may appeal with a description of intended use and safeguards.
