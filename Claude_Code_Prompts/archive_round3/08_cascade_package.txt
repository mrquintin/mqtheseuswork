## Your task
You are an implementation agent. Your job is to create a self-contained `noosphere/noosphere/cascade/` package implementing a typed, cycle-free, append-only graph of evidence between artifacts/chunks/claims/principles/conclusions. Every edge in the graph is attributed to a specific method_invocation_id from the ledger. You will register a post-hook with the method decorator so that ported methods automatically emit edges declared via `emits_edges` in their `@register_method` call.

CascadeGraph.add_edge (exact contract)
def add_edge(self, *, src: str, dst: str, relation: CascadeEdgeRelation,
             method_invocation_id: str, confidence: float, unresolved: bool = False) -> str:
    # 1. Validate method_invocation_id exists in Store.method_invocations; else CascadeEdgeOrphanError.
    # 2. If relation in {supports, refutes}, check for a non-retracted conflicting edge at same (src, dst)
    #    in the complementary relation; raise CascadeEdgeConflictError if present.
    # 3. If relation == depends_on, check that adding this edge does not create a cycle in depends_on
    #    subgraph (incremental BFS from dst forward looking for src); raise CascadeCycleError if cycle.
    # 4. Build CascadeEdge, call Store.insert_cascade_edge, return edge_id.

Hook behavior (hooks.py)
On import:
  register_post_hook("cascade.emit_edges", _emit_edges)

def _emit_edges(spec, inv, input, result):
    for relation in spec.emits_edges:
        field_name = f"{relation}_targets"
        if hasattr(result, field_name):
            targets = getattr(result, field_name)
            for dst in targets:
                CascadeGraph(store).add_edge(src=input.primary_src,       # or input.node_id, depending on method
                                             dst=dst, relation=relation,
                                             method_invocation_id=inv.id,
                                             confidence=getattr(result, "confidence", 1.0))

The exact `primary_src` resolution is spec-specific; document the convention in cascade/hooks.py docstring. For methods whose input does not carry a single source node (e.g. aggregators), require the method output itself to contain `emitted_edges: list[{src,dst,relation,confidence}]` instead and follow that structure.
