## Tests
In `tests/test_store_round3.py`, use an in-memory SQLite fixture and assert:
1. For every new table, an insert followed by a get returns the same object.
2. `append_ledger_entry` rejects an entry whose `prev_hash` does not match the current tail, raising `LedgerChainError`.
3. `append_ledger_entry` succeeds when `prev_hash` matches; the new tail equals the inserted entry.
4. `insert_cascade_edge` raises `CascadeEdgeOrphanError` if the referenced method_invocation_id does not exist.
5. `insert_cascade_edge` raises `CascadeEdgeConflictError` if you insert a `supports` edge where a non-retracted `refutes` edge already exists between the same (src, dst).
6. `retract_cascade_edge` sets `retracted_at` and makes `iter_cascade_edges(include_retracted=False)` exclude it.
7. Migration upgrade → downgrade → upgrade is a no-op (diff before and after is empty).
8. Existing round-2 tests in `tests/test_store.py` still pass.
