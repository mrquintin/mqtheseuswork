## Step-by-step

### 1. `extractors/base.py` — the contract

```python
from __future__ import annotations
from dataclasses import dataclass, field
from typing import Protocol, Literal

BinarySource = Literal["db", "supabase", "local"]


@dataclass(frozen=True)
class TextContent:
    text: str


@dataclass(frozen=True)
class BinaryContent:
    data: bytes
    mime: str
    filename: str
    source: BinarySource


UploadContent = TextContent | BinaryContent


@dataclass(frozen=True)
class ExtractedText:
    text: str
    source_format: str          # e.g. "text/plain", "audio/mp4", "application/pdf"
    extraction_method: str      # e.g. "passthrough", "faster-whisper", "pypdf"
    warnings: list[str] = field(default_factory=list)


class ExtractionFailed(Exception):
    """Extractor ran but could not produce text (corrupt file, empty audio, etc.)."""


class UnsupportedMimeType(Exception):
    """No handler registered for this MIME family."""


class Extractor(Protocol):
    name: str
    mime_prefixes: tuple[str, ...]  # e.g. ("audio/",), ("application/pdf",)

    def extract(self, content: BinaryContent) -> ExtractedText: ...
```

### 2. `extractors/text_extractor.py` — passthrough + structured text handling

For binary, attempt UTF-8, fall back to latin-1 with a warning, last resort raise `ExtractionFailed`. JSON pretty-prints before returning (so claim extraction sees indented text it can tokenize). ### 5. `extractors/_content_fetcher.py` — Upload.filePath → bytes

Given an Upload row, return `UploadContent`:

- If `textContent` is non-empty → `TextContent(textContent)`.
- Else, read `filePath`:
  - Starts with `s3://` or `supabase://` → pull from Supabase Storage using `SUPABASE_URL` + `SUPABASE_SERVICE_ROLE_KEY` + the bucket path. Use a signed download URL with a 10-minute TTL; stream to a temp file, return bytes with `source="supabase"`.
  - Starts with `file://` or looks like an absolute local path that exists → read from disk, `source="local"`.
  - Otherwise → raise `ExtractionFailed("Upload {id}: filePath '{path}' cannot be resolved; expected supabase:// or local absolute path.")`.
- Size guard: reject > `NOOSPHERE_MAX_UPLOAD_BYTES` (default 500 MiB; env overridable) — large enough for the 194.8 MB m4a that triggered this work, small enough to reject accidental 4 GB video drops.

No streaming into the extractor — the extractors receive `bytes` so they are trivially testable. If a 500 MiB upload becomes common, a later prompt can add streaming. Ship the simple thing first.

### 6. `extractors/__init__.py` — registry

```python
from noosphere.extractors.base import (
    ExtractedText, Extractor, ExtractionFailed, UnsupportedMimeType,
    BinaryContent, TextContent, UploadContent,
)
from noosphere.extractors.text_extractor import TextExtractor
from noosphere.extractors.audio_extractor import AudioExtractor
from noosphere.extractors.pdf_extractor import PdfExtractor

DEFAULT_EXTRACTORS: tuple[Extractor, ...] = (
    TextExtractor(),
    AudioExtractor(),
    PdfExtractor(),
)

def dispatch(content, extractors=DEFAULT_EXTRACTORS) -> ExtractedText:
    """Find the first extractor whose mime_prefixes match content.mime (BinaryContent)
    or return TextContent unchanged wrapped in ExtractedText."""
    from noosphere.extractors.base import TextContent, BinaryContent
    if isinstance(content, TextContent):
        return ExtractedText(text=content.text, source_format="text/plain",
                             extraction_method="passthrough")
    for ex in extractors:
        if any(content.mime.startswith(p) for p in ex.mime_prefixes):
            return ex.extract(content)
    raise UnsupportedMimeType(
        f"no extractor for mime={content.mime!r}; "
        f"supported families: {', '.join(p for ex in extractors for p in ex.mime_prefixes)}"
    )
```
