### `opinion_generator.py`

```python
from enum import Enum

class OpinionOutcome(str, Enum):
    PUBLISHED = "PUBLISHED"
    ABSTAINED_BUDGET = "ABSTAINED_BUDGET"
    ABSTAINED_INSUFFICIENT_SOURCES = "ABSTAINED_INSUFFICIENT_SOURCES"
    ABSTAINED_NEAR_DUPLICATE = "ABSTAINED_NEAR_DUPLICATE"
    ABSTAINED_CITATION_FABRICATION = "ABSTAINED_CITATION_FABRICATION"

async def generate_opinion(store, event_id: str, *, budget) -> OpinionOutcome:
    """
    Run retrieve_for_event → call Haiku with strict JSON output schema →
    validate citations are verbatim substrings → write EventOpinion +
    OpinionCitations, or ABSTAIN with reason.

    Up to 2 retries on JSON parse failure or citation-fabrication.
    Persistent failure → ABSTAINED_CITATION_FABRICATION.
    """
```

System prompt requirements (live in `_prompts/opinion_system.md`, NOT inlined):
- "You opine ONLY from the retrieved Theseus sources below. If they don't support a position, return stance=ABSTAINED."
- "Every citation's `quoted_span` must be a verbatim substring of the cited source."
- Strict JSON schema for the response: `{stance, confidence, headline, body_markdown, uncertainty_notes, citations, topic_hint}`.

Citation validator: for each `quoted_span`, look up the source, confirm `quoted_span in source.text` (exact substring, no whitespace normalization). On failure, retry once with a corrective system message; on second failure, abstain.

### `followup.py`

```python
async def answer_followup(
    store, opinion_id: str, session_id: str, user_question: str, *, budget,
) -> AsyncIterator[FollowupAnswerChunk]:
    """
    1. Re-retrieve from the WHOLE Noosphere using user_question (NOT the
       opinion's existing citations — fresh retrieval per question).
    2. Wrap user_question with PromptSeparator delimiters.
    3. Stream Haiku response: yield meta → token chunks → done.
    4. Validate citations same as opinion_generator. Drop hallucinated ones
       silently from the response (don't surface them).
    5. Persist user + assistant FollowUpMessages.
    """

@dataclass
class FollowupAnswerChunk:
    kind: Literal["meta", "token", "citation", "done"]
    text: str | None
    citation: dict | None
```

Rate limits enforced inline:
- `RATE_LIMIT_PER_FINGERPRINT_PER_DAY = 20` — 21st request from the same fingerprint same UTC day → 429-style outcome (the FastAPI layer translates).
- `RATE_LIMIT_PER_SESSION = 8` — 9th message in a session → blocked.
- Min interval between messages enforced server-side: 2 seconds.

## Tests

`noosphere/tests/test_opinion_generator.py`:
- Happy path: scripted Haiku reply with valid citations → PUBLISHED.
- Citation `quoted_span` not a substring → retry; second failure → ABSTAINED_CITATION_FABRICATION; no row written.
- Budget exhausted → ABSTAINED_BUDGET; no Anthropic call made.
- ABSTAINED stance returned by LLM → no opinion written, event status ABSTAINED.
