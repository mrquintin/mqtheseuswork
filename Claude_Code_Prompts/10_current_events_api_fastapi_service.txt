### Routes

| Method | Path | Notes |
|---|---|---|
| GET | `/v1/currents` | List opinions, query params: `since`, `until`, `topic`, `stance`, `limit ‚â§ 50` |
| GET | `/v1/currents/{id}` | Single opinion + citations |
| GET | `/v1/currents/{id}/sources` | Full source detail (Conclusion or Claim text) for each citation |
| GET | `/v1/currents/stream` | SSE ‚Äî emits new PublicOpinion frames as they arrive; heartbeat every 15s |
| POST | `/v1/currents/{id}/follow-up` | SSE ‚Äî accepts `{question, session_id?}`; streams meta ‚Üí tokens ‚Üí citation frames ‚Üí done |
| GET | `/v1/currents/{id}/follow-up/{session_id}/messages` | Paginated message history |
| GET | `/healthz` | `{ok: true}` ‚Äî for load balancers |
| GET | `/readyz` | Checks DB reachable + scheduler liveness file fresh |
| GET | `/metrics` | Prometheus text format |

### `PublicOpinion` schema

Strip internal fields (`promptTokens`, `completionTokens`, `clientFingerprint`, raw `revokedReason` text ‚Äî keep only `revokedSourcesCount` aggregate). The wire shape is what theseus-codex's route handlers proxy through unchanged.

### `event_bus.py`

```python
class OpinionBus:
    def __init__(self): self._subscribers = []
    async def publish(self, opinion_dict): ...
    async def subscribe(self) -> AsyncIterator[dict]: ...
    def feed_client_count(self) -> int: return len(self._subscribers)
    def followup_client_count(self) -> int: ...
```

In-memory, per-process. Single-worker invariant.

### `publisher.py`

`OpinionTailer` ‚Äî polls DB every 2s for opinions with `generatedAt > cursor`, publishes to bus. Started in FastAPI lifespan. This is what makes new opinions land in connected SSE clients without pub/sub infra.

### `rate_limit.py`

Sliding-window: 30 requests / IP / minute on the read endpoints; 20 follow-up POSTs / fingerprint / day; 8 messages / session. Returns 429 with JSON body indicating reason.

Read `CURRENTS_CORS_ORIGINS` env (comma-separated). The merged Codex deploy origin (e.g. `https://mqtheseuswork-qiw6.vercel.app`) goes here. No wildcard allowed.
