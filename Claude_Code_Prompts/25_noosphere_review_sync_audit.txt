## The problem

When a founder resolves a review item, the Codex DB is always updated. But the Noosphere sync (via `pushReviewResolutionToNoosphere`) can fail silently:
- The API returns a 200 with a `warning` field, but the client-side `ReviewQueue` doesn't display warnings (it only checks `res.ok`).
- The failure is not recorded in the `AuditEvent` table or anywhere else persistent.
- There is no retry mechanism — if the sync fails, the Noosphere store permanently diverges.
- The batch resolution route (if it exists) does fire-and-forget with `.catch(() => {})`.

This means the Noosphere and Codex databases can silently drift apart on review resolutions with no audit trail.

## Your task

1. Log Noosphere sync failures as `AuditEvent` records so they're visible and queryable.
2. Surface sync warnings in the client UI.
3. Add a retry mechanism for failed syncs.

4. **Create a retry endpoint** at `api/review/retry-sync/route.ts`:
   ```typescript
   export async function POST(req: Request) {
     const founder = await getFounder();
     if (!founder) return NextResponse.json({ error: "Unauthorized" }, { status: 401 });

     const { reviewItemId } = await req.json() as { reviewItemId: string };
     const item = await db.reviewItem.findUnique({ where: { id: reviewItemId } });
     if (!item || !item.noosphereId || item.status !== "done" || !item.humanVerdict) {
       return NextResponse.json({ error: "Item not eligible for retry" }, { status: 400 });
     }

     const result = await pushReviewResolutionToNoosphere({
       reviewId: item.noosphereId,
       verdict: item.humanVerdict as "cohere" | "contradict" | "unresolved",
       overrule: item.humanOverrule,
       aggregatorVerdict: item.aggregatorVerdict,
       founderId: founder.noosphereId || founder.id,
       note: item.resolutionNote,
     });

     if (result.ok) {
       // Log successful retry
       await db.auditEvent.create({
         data: {
           organizationId: founder.organizationId,
           founderId: founder.id,
           action: "noosphere_sync_retried",
           detail: JSON.stringify({ reviewItemId, success: true }),
         },
       });
       return NextResponse.json({ ok: true });
     }

     return NextResponse.json({ ok: false, error: result.stderr }, { status: 500 });
   }
   ```
