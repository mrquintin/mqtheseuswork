COMMAND SURFACES

review.py — run <conclusion_id>, rebuttals-pending, reviewer-calibration.

decay.py — status [--tier firm|founder|open] [--freshness fresh|aging|stale|retired], revalidate <object_id>, schedule, retire <object_id> --reason ...
