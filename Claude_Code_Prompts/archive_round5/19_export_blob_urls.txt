## The problem

The `downloadHref` function creates data: URIs by encoding the entire dataset into the URL. For large datasets (hundreds of peer reviews, decay records, etc.), this creates URLs that are megabytes long. This has multiple failure modes:
- Browsers have URL length limits (2MB in Chrome, varying in others)
- The URL is embedded in the server-rendered HTML, bloating page size
- `encodeURIComponent` on a large string is CPU-intensive and blocks the event loop

## Your task

Replace the data: URI approach with a client-side Blob URL approach, or with a server-side download endpoint. Since the pages are server components, the cleanest solution is a thin client component that generates Blob URLs on the client side.

## Step-by-step

1. **Create `DownloadButton.tsx`** — a client component that generates a Blob URL on click:
   ```tsx
   "use client";
   import { useCallback } from "react";

   export default function DownloadButton({
     data,
     filename,
     mime,
     label,
     className,
     style,
   }: {
     data: string;
     filename: string;
     mime: string;
     label: string;
     className?: string;
     style?: React.CSSProperties;
   }) {
     const handleClick = useCallback(() => {
       const blob = new Blob([data], { type: `${mime};charset=utf-8` });
       const url = URL.createObjectURL(blob);
       const a = document.createElement("a");
       a.href = url;
       a.download = filename;
       a.click();
       URL.revokeObjectURL(url);
     }, [data, filename, mime]);

     return (
       <button type="button" onClick={handleClick} className={className} style={style}>
         {label}
       </button>
     );
   }
   ```

2. **Replace all `<a href={downloadHref(...)}>` patterns** with `<DownloadButton>`. For example, in the peer review page:
   ```tsx

   <a href={downloadHref(csvData, "text/csv")} download="peer-review.csv" className="btn">Download CSV</a>

   // After:
   <DownloadButton data={csvData} filename="peer-review.csv" mime="text/csv" label="Download CSV" className="btn" style={{ fontSize: "0.65rem" }} />
   ```
