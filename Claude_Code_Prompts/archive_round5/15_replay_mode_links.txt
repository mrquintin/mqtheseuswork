## The problem

In temporal replay mode (`/conclusions?asOf=2025-01-15`), conclusions render without `<Link>` wrappers. The user can see the replay-consistent conclusions but cannot click into any of them to view details, provenance, or peer reviews. The normal-mode list links each conclusion to `/conclusions/[id]`; the replay-mode list does not.

## Your task

Wrap replay-mode conclusion items in clickable links, the same way the normal mode does. The detail page itself is not replay-aware (it always shows current state), so add a visual indicator that the user is leaving replay mode when they click through.

