### 3. `_whisper_local.py` — faster-whisper wrapper

```python
from __future__ import annotations
import logging, os, tempfile
from pathlib import Path

_MODEL_CACHE = {}

def _get_model(name: str):
    """Cache WhisperModel instances per-process. First load is ~5s; reuse."""
    if name not in _MODEL_CACHE:
        from faster_whisper import WhisperModel
        # compute_type="int8" — ~2× faster on CPU with negligible quality loss
        # for English speech. Override via NOOSPHERE_WHISPER_COMPUTE_TYPE.
        _MODEL_CACHE[name] = WhisperModel(
            name,
            device="cpu",
            compute_type=os.environ.get("NOOSPHERE_WHISPER_COMPUTE_TYPE", "int8"),
        )
    return _MODEL_CACHE[name]


def transcribe_file(path: Path, *, language: str | None = "en") -> str:
    model_name = os.environ.get("NOOSPHERE_WHISPER_MODEL", "small.en")
    model = _get_model(model_name)
    # beam_size=1, vad_filter=True gives a good speed/quality trade-off
    # for the kinds of long meeting recordings Theseus founders record.
    segments, info = model.transcribe(
        str(path), language=language, beam_size=1, vad_filter=True,
    )
    return " ".join(seg.text.strip() for seg in segments if seg.text).strip()
```

Timings baseline on an M-series Mac, `small.en`, int8, 60-min English meeting: ~6-8 min wall-clock. Document this in the module docstring — founders running this command should know the rough cost.

### 4. `_whisper_openai.py` — cloud fallback with chunking

OpenAI Whisper API caps individual files at 25 MiB. The 194.8 MB m4a must be split. Use `mutagen` + raw byte slicing on the container is not safe for arbitrary codecs; use `ffmpeg` (must be installed on the runner) to split into 10-minute .mp3 chunks, transcribe sequentially, concatenate. If `ffmpeg` is not available, raise with a clear install hint.

```python
def transcribe_file(path: Path, *, api_key: str, language: str = "en") -> str:
    meta = probe(path)
    if meta.size_bytes <= 24 * 1024 * 1024 and meta.duration_seconds <= 25 * 60:
        return _transcribe_single(path, api_key, language)
    return _transcribe_chunked(path, api_key, language)
```

Concatenation is done with a single space between chunks — do NOT insert chunk markers; they pollute downstream claim extraction.

### 5. `audio_extractor.py` — fill in `extract`

```python
import os, tempfile
from pathlib import Path
from noosphere.extractors.base import (
    Extractor, BinaryContent, ExtractedText, ExtractionFailed,
)

class AudioExtractor:
    name = "audio"
    mime_prefixes = ("audio/",)

    def extract(self, content: BinaryContent) -> ExtractedText:
        warnings: list[str] = []
        # Persist to a temp file — faster-whisper and openai both want a path.
        suffix = "." + content.filename.rsplit(".", 1)[-1] if "." in content.filename else ".bin"
        with tempfile.NamedTemporaryFile(suffix=suffix, delete=False) as tmp:
            tmp.write(content.data)
            tmp_path = Path(tmp.name)
        try:
            force_openai = os.environ.get("NOOSPHERE_FORCE_OPENAI_WHISPER") == "1"
            if not force_openai:
                try:
                    from noosphere.extractors._whisper_local import transcribe_file as local_transcribe
                    text = local_transcribe(tmp_path)
                    return ExtractedText(
                        text=text, source_format=content.mime,
                        extraction_method="faster-whisper",
                        warnings=warnings,
                    )
                except Exception as e:
                    warnings.append(f"faster-whisper failed: {type(e).__name__}: {e}")
            api_key = os.environ.get("OPENAI_API_KEY", "").strip()
            if not api_key:
                raise ExtractionFailed(
                    "audio transcription failed: faster-whisper unavailable "
                    "and OPENAI_API_KEY not set. Install with "
                    "`pip install 'noosphere[audio]'` or export OPENAI_API_KEY."
                )
            from noosphere.extractors._whisper_openai import transcribe_file as openai_transcribe
            text = openai_transcribe(tmp_path, api_key=api_key)
            return ExtractedText(
                text=text, source_format=content.mime,
                extraction_method="openai-whisper-1",
                warnings=warnings,
            )
        finally:
            try: tmp_path.unlink()
            except OSError: pass
```

## Prohibitions

- Do NOT hold the faster-whisper model in a module-level global that gets pickled or loaded at import time. Lazy load only inside `_get_model` — test runs must not trigger model downloads.
- Do NOT chunk at the byte level. AAC frames don't align on byte boundaries. Use ffmpeg for the OpenAI chunking path.
- Do NOT add chunk separators to the concatenated text. Whisper's output is already prose; markers break downstream NLP.
- Do NOT let `ExtractionFailed` leak the raw OpenAI or httpx exception string — wrap it with a clean remediation message. The raw exception goes to logs.
- Do NOT call `faster_whisper.WhisperModel` with `device="auto"`. Force `cpu`; GPU auto-detection has historically crashed on laptops without CUDA.
- Do NOT add a `max_duration` hard cap in this prompt. Long meetings are a legitimate use case; a budget system belongs in a later prompt if it's needed at all.
