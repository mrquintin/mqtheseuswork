## The problem

Contradictions currently persist forever once detected. There is no way for a founder to:
- **Dismiss** a false positive (the system flagged a contradiction that isn't one)
- **Resolve** a genuine contradiction by recording how it was addressed (e.g., "I updated my position" or "these claims are compatible in context X")
- **Hide** contradictions they've already reviewed from the active list

The founder needs a workflow where contradictions can be triaged, similar to how issues are triaged in a bug tracker: review → resolve with a note, or dismiss as false positive.

1. **Add resolution fields to the `Contradiction` model** in `schema.prisma`:
   ```prisma
   model Contradiction {
     // ... existing fields ...

     // ── Resolution tracking ────────────────────────────────────────────
     // A contradiction starts as "active". A founder can resolve it
     // (acknowledged and addressed) or dismiss it (false positive).
     // Resolved/dismissed contradictions are hidden from the main view
     // but preserved for audit.
     status       String    @default("active")  // active | resolved | dismissed
     resolution   String?   // founder's note explaining the resolution
     resolvedById String?
     resolvedAt   DateTime?

     resolvedBy Founder? @relation("ContradictionResolver", fields: [resolvedById], references: [id])

     // ... existing relations ...
   }
   ```
   Add the reverse relation on `Founder`:
   - `resolvedContradictions Contradiction[] @relation("ContradictionResolver")`

3. **Create the API route `PATCH /api/contradictions/[id]`:**
   - Authenticate the founder via `requireTenantContext()`.
   - Accept `{ action: "resolve" | "dismiss", resolution?: string }`.
   - Verify the contradiction belongs to the founder's organization.
   - For `resolve`: set `status = "resolved"`, `resolution`, `resolvedById`, `resolvedAt = now()`.
   - For `dismiss`: set `status = "dismissed"`, `resolution` (optional note), `resolvedById`, `resolvedAt = now()`.
   - Create an `AuditEvent` recording the action.
   - Return the updated contradiction.
