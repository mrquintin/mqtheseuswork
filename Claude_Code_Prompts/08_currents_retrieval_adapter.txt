**Sequence:** prompt 8 of 22 (Wave 3 opener). Depends on 05–07. Report back with: top-k default chosen, Stage A/B counts on a real event, whether the existing HybridRetriever needed any modification.

**Why this prompt exists.** Prompt 09 (the opinion generator) needs source material to ground an opinion in. The existing `HybridRetriever` in `noosphere/noosphere/retrieval.py` does BM25 + dense embedding retrieval. We need a thin adapter that:

- Stage A: looks up *Conclusions* directly (founder-canonical positions; the strongest grounding).
- Stage B: looks up *Claims* via HybridRetriever (more granular, broader coverage).
- Filters out claims that are subsumed by an already-retrieved Conclusion (cosine > 0.85 to a Conclusion hit).
- Filters out `ClaimOrigin.EXTERNAL` and `ClaimOrigin.ADVERSARIAL` (only firm-internal sources ground opinions).
- Returns a unified `EventRetrievalHit` list ordered by relevance score.
