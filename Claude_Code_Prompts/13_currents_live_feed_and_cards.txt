## Your task

Build `/currents/page.tsx` (server component, seeded), `FeedClient.tsx` (client; consumes `useLiveOpinions`), `OpinionCard.tsx` (the visual element), `LiveBanner.tsx` (the "live" / "reconnecting…" indicator), and small helpers `safeMarkdown.ts` + `relativeTime.ts`.

### `OpinionCard` shape

- Border-left in the stance color (`--currents-stance-{agrees|disagrees|complicates|abstained}`).
- Top metadata row: stance pill, confidence band ("low/mid/high"), topic hint, age (relative time).
- Headline (EB Garamond, serif, 1.15rem), links to `/currents/{id}`.
- Body markdown rendered safely (paragraphs, italic, bold, inline code, links — nothing else; no raw HTML).
- Uncertainty notes in italic amber if any.
- Source strip at bottom: up to 3 citation chips ("⸺ conclusion" / "⸺ claim"), each linking to `/currents/{id}#src-{source_id}`. "+N more" if exceeded.
- "Ask a follow-up →" link in gold to `/currents/{id}#ask`.
- Author handle + external source link in muted text.

### `safeMarkdown.ts`

Pure function: `renderSafeMarkdown(md: string): React.ReactNode`. Hand-rolled is fine — body is ≤ 800 chars by contract. Aggressively escape `<`, `>`, `&`. Links get `rel="noopener nofollow ugc" target="_blank"`. Reject any `<script>` content.

### `relativeTime.ts`

`relativeTime(iso, now=Date.now()): string` — "5s ago", "2m ago", "3h ago", "8d ago".

### `LiveBanner.tsx`

```tsx
<div className="…">
  <span className={connected ? "currents-pulse" : ""}
        style={{ background: connected ? undefined : "var(--currents-muted)" }} />
  <span>{connected ? "live" : "reconnecting…"}</span>
  {lastOpinionAt && <span>· last update {relativeTime(lastOpinionAt)}</span>}
</div>
```

### `FeedClient.tsx`

Reads the seed prop, hydrates via `useLiveOpinions`, renders `<LiveBanner>` then a list of `<OpinionCard>`. New opinions get `currents-fade-in` animation (the keyframe from prompt 12).

Empty state: parchment-bordered card explaining "No opinions yet. The firm abstains when its knowledge base does not support a position."

### `/currents/page.tsx` (server)

```tsx
export const dynamic = "force-dynamic";

export default async function CurrentsPage() {
  let seed: PublicOpinion[] = [];
  try {
    const resp = await listCurrents({ limit: 20 });
    seed = resp.items;
  } catch (err) {
    console.error("currents_seed_fetch_failed", err);
    // FeedClient handles empty seed; no need to throw
  }
  return <FeedClient seed={seed} />;
}
```
