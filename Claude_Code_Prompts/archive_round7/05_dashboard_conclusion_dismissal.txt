## The problem

The dashboard displays the 8 most recent conclusions with no ability to dismiss, hide, or remove them from the homepage view. The founder described wanting to be able to "delete conclusions from the home dashboard itself at the very home page." This doesn't mean permanent deletion (that's prompt 04's deletion request system) — it means dismissing a conclusion from the dashboard so it no longer clutters the homepage.

This is a per-founder UI preference: founder A dismissing a conclusion from their dashboard should not affect founder B's view.

## Step-by-step

1. **Add a `DashboardDismissal` model** to `schema.prisma`. This is a lightweight join table recording which founder has dismissed which conclusion from their dashboard:
   ```prisma
   model DashboardDismissal {
     id           String   @id @default(cuid())
     founderId    String
     conclusionId String
     createdAt    DateTime @default(now())

     founder    Founder    @relation(fields: [founderId], references: [id])
     conclusion Conclusion @relation(fields: [conclusionId], references: [id], onDelete: Cascade)

     @@unique([founderId, conclusionId])
     @@index([founderId])
     @@index([conclusionId])
   }
   ```
   Add the reverse relations:
   - On `Founder`: `dashboardDismissals DashboardDismissal[]`
   - On `Conclusion`: `dashboardDismissals DashboardDismissal[]`

2. **Generate and apply the Prisma migration:**
   ```bash
   npx prisma migrate dev --name add_dashboard_dismissals
   ```

3. **Create the API route `POST /api/dashboard-dismissals`:**
   - Authenticate the founder via `requireTenantContext()`.
   - Accept `{ conclusionId }`.
   - Verify the conclusion exists and belongs to the founder's organization.
   - Upsert to prevent duplicates (use the `@@unique` constraint).
   - Return `{ ok: true }`.

4. **Modify the dashboard conclusions query** in `dashboard/page.tsx` to exclude dismissed conclusions for the current founder:
   ```typescript
   const conclusions = await db.conclusion.findMany({
     where: {
       organizationId: tenant.organizationId,
       dashboardDismissals: {
         none: { founderId: tenant.founderId },
       },
     },
     orderBy: { createdAt: "desc" },
     take: 8,
   });
   ```
